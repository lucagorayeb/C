# Classes de Armazenamento

## Variáveis automáticas

As palavras auto e register são utilizadas para declarar 
variáveis com tempo de armazenamento automático.

As variáveis automáticas são criadas quando o bloco onde
foram criadas é acionado e são destruidas quando o bloco 
é encerrado

Variáveis automaticas raramente são declaradas com o 
os seus respectivos nomes, pois o compilador entende 
todas as váriaveis como automaticas por padrão.

## Variáveis estatícas

As palavras extern e static são usadas para declarar 
identificadores para variáveis e funções estatícas.

As variáveis globais e funções são exemplos padrões de 
armazenamento estático. As variáveis locais estáticas 
locais são antecedidas da palavra static.

Globais e funções pertencem a classe extern por padrão.
